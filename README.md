# O2O
这是一个校园o2o商铺,分为两个版本

+ v0.01（ssm）

+ v0.02(springboot)

## 技术栈

+ **前端：**前端UI库 SUI Mobile／ jQuery／ Ajax

+ **后端：**SSM／Spring Boot／图片开源工具 Thumbnailator/ 
  验证码组件Kaptcha／作业调度框架 Quartz/Shell

+ **平台：**微信测试号

+ **数据库：**MySQL

+ **缓存：**Redis

+ **部署：**阿里云

+ **技术亮点：**MySQL主从同步/项目读写分离/框架迁移/代码重构
